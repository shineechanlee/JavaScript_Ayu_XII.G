Percabangan pada JavaScript

Percabangan adalah struktur kontrol untuk menentukan alur program berdasarkan kondisi tertentu.

1. if

Menjalankan kode jika kondisi bernilai true.

if(kondisi){ aksi; }

2. if / else

Dua pilihan: kondisi benar atau salah.

if(kondisi){ aksi1; } else { aksi2; }

3. if / else if / else

Digunakan untuk lebih dari dua kondisi.

if(k1){...}
else if(k2){...}
else {...}

4. switch / case

Alternatif untuk banyak pilihan nilai.

switch(nilai){
case "1": aksi; break;
default: aksi; }

5. Operator Ternary

Bentuk singkat dari if/else.

var hasil = (kondisi) ? "Benar" : "Salah";

6. Nested (Bersarang)

Percabangan di dalam percabangan lain.

if(k1){
  if(k2){ aksi; }
}

Bonus: Operator Logika

Menggunakan AND (&&) atau OR (||) untuk menyederhanakan kondisi.

if(user == "admin" && pass == "123"){ ... }
