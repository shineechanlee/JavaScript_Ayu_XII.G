Operator pada JavaScript

Operator adalah simbol yang digunakan untuk melakukan operasi pada nilai atau variabel dalam program.

1. Operator Aritmatika

Digunakan untuk perhitungan matematika. Contoh: +, -, *, /, %, **.

var hasil = a + b;

2. Operator Penugasan

Digunakan untuk memberi atau memperbarui nilai variabel. Contoh: =, +=, -=, *=, /=, %=.

nilai += 5;

3. Operator Perbandingan

Membandingkan dua nilai dan menghasilkan true atau false. Contoh: ==, ===, !=, >, <, >=, <=.

var cek = a === b;

4. Operator Logika

Digunakan untuk menggabungkan kondisi. Contoh: && (AND), || (OR), ! (NOT).

if(a > 5 && b < 10){ ... }

5. Operator Bitwise

Operasi berbasis bilangan biner. Contoh: &, |, ^, ~, <<, >>.

var hasil = a & b;

6. Operator Ternary

Bentuk singkat dari if/else.

var hasil = (kondisi) ? "Benar" : "Salah";

Keenam operator ini penting untuk memahami logika dan perhitungan dalam pemrograman JavaScript.
